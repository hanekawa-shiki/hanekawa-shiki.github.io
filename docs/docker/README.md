**Build,Ship and Run Any App,Anywhere**

`Docker`是个划时代的开源项目，它彻底释放了计算虚拟化的威力，极大提高了应用的维护效率，降低了云计算应用开发的成本！`Docker`之所以发展如此迅速，也是因为它给出了一个标准化的解决方案-----系统平滑移植，容器虚拟化技术 。
